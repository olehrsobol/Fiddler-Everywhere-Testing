# Fiddler-Everywhere-Testing

This repository contains examples of traffic analysis and manipulation using *Fiddler Everywhere*.

Used Fiddler to intercept and modify HTTP/HTTPS traffic for testing purposes from the <strong><a href="https://demowebshop.tricentis.com/">Demo Web Shop by Tricentis</a></strong>.

> <strong>Included Tasks:</strong>
<ol>
    <li>Modify request (change product quantity)</li>
    <li>Modify response (change price)</li>
    <li>Simulate HTTP 500 error</li>
    <li>Redirect requests (Prod → QA)</li>
    <li>Full response replacement</li>
</ol>
